# Histórico EJA

Sistema para gestão de históricos escolares e certificados para Educação de Jovens e Adultos (EJA).

## Funcionalidades
- Importação de alunos via Excel/CSV
- Edição de notas e frequências
- Geração de PDF (Certificado e Histórico)
- Controle de acesso de operadores
- Inativação de usuários
